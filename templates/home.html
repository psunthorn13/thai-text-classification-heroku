Text Classification in Thai Language

The objective of this project is to create end-to-end app that uses NLP model for Thai Language to classify message category so that users can take action accordingly. We uses dataset from Thai text classification workshop (https://www.youtube.com/watch?v=DRWXKUAEa4s&list=WL&index=61), which is the sample data encoded from call center voice from the telecommunication company(True). There are 7 actions/ classes, which will be predicted namely enquire, report,cancel, buy, activate,request, change
